## Como usar esta biblioteca

### Para o Designer Agent
```
Ao criar UI, consulte sites desta categoria:
- Landing Pages: para estrutura e CTAs
- SaaS: para dashboards e interfaces de produto
- Portfolios: para layouts criativos e tipografia
```

### Para o UXer Agent
```
Ao analisar UX, estude:
- Fluxos de onboarding em SaaS
- Hierarquia visual em Landing Pages
- Microinterações em Portfolios
```
